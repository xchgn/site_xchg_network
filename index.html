features

XCHG — the fuel of the network
Every program in the XCHG network has its own address. One XCHG token activates this address for 24 hours.
One token = one address
The economy is simple and transparent: 1 XCHG powers one address in the network for a full day.
Direct token utility
XCHG is not just a speculative asset but a real resource. The more tokens you hold, the more addresses you can operate daily.
Programs need tokens
Each application must hold its own XCHG to function every day. This creates stable demand and a strong network economy.
Fair rewards for contributors
The XCHG network automatically distributes rewards to application developers and node operators who provide routing for the traffic.
Built-in incentives
Every transaction in the network generates fair compensation: developers receive a share for building applications, while router owners are rewarded for forwarding data.
Sustainable economy
With automated payouts, XCHG ensures that everyone who contributes to the network — from developers to infrastructure providers — gets rewarded transparently and instantly.
DAO Governance
The XCHG network is governed through a DAO, where voting power is proportional to the size of the stake. The architecture is fully decentralized, covering everything from protocol upgrades to reward distribution rules.
Access to external services
The XCHG token can also be used to access APIs of various services — from blockchain oracles to weather data providers.

applications

IoT
XCHG enables secure, low-latency data exchange between IoT devices such as ESP32 or Arduino, without relying on centralized servers. Metrics, sensor readings, and control commands can be transmitted directly between devices using the decentralized XCHG network. This approach simplifies cloud integration, reduces operational costs, and ensures data privacy — all while maintaining seamless connectivity across diverse environments.
Deep API
With XCHG, developers can expose API endpoints located behind NAT or firewalls — without opening ports or configuring routers. The XCHG network routes encrypted requests directly to the target device, allowing any application or service to act as an accessible API provider from anywhere. This makes it possible to publish private, peer-hosted APIs with zero infrastructure and full end-to-end security.
DIY Projects
XCHG allows makers and hobbyists to connect PCs and custom-built devices without needing a personal server or a fixed IP address. It provides a simple way to establish secure, peer-to-peer communication between your projects — whether it’s a home automation setup, a robot, or a custom sensor module. With XCHG, even small DIY systems gain the connectivity power of the cloud, but with no central point of control.
Remote Control
XCHG enables secure remote control of devices and computers without relying on VPNs or external servers. Commands and data streams are delivered through the decentralized XCHG network, ensuring low latency and full encryption. Whether you need to control an industrial controller, a remote PC, or a microcontroller board, XCHG makes it possible to reach any device, anywhere.
Monitoring
XCHG provides a decentralized solution for monitoring systems and devices without the need for centralized servers. Data from sensors, applications, or infrastructure can be securely transmitted through the XCHG network to monitoring dashboards or alerting systems. This approach ensures data integrity and privacy while enabling real-time insights into system performance from anywhere in the world.
File Exchange
XCHG enables secure, peer-to-peer file exchange without relying on centralized services. Files can be shared directly between devices through the decentralized XCHG network, ensuring end-to-end encryption and data integrity. This makes it easy to transfer documents, media, or any other files between users or devices, while maintaining full control over access and distribution.

Token XCHG

The XCHG token (XCHG) is the core utility asset of the XCHG network, built on the SUI blockchain. It powers all interactions within the ecosystem — including traffic payments, node rewards, and developer incentives. The total supply is 10,000,000,000 XCHG, permanently capped to ensure long-term stability and predictable economics.
